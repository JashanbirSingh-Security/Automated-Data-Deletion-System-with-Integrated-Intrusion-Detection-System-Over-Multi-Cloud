# Automated-Data-Deletion-System-with-Integrated-Intrusion-Detection-System-Over-Multi-Cloud

Project Title: Automated Data Deletion System in the Event of Intrusion Detection

Project Overview: This project proposes a cloud security mechanism that automatically deletes sensitive or confidential data in an event of intrusion is detected. The objective is to reduce the risk of unauthorized access and data theft in cloud environments.

Research Type: Qualitative Research Study

Key Concepts: Cloud Computing, Data Security, Intrusion Detection System, Automated Response Systems

Publication: DOI: 10.26438/ijcse/v12i7.3340
